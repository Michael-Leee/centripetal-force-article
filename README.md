# 向心力详解

一篇系统介绍向心力（Centripetal Force）的 LaTeX 学术文章。

## 内容涵盖

- 向心力定义与物理本质
- 从牛顿第二定律推导向心力公式
- 向心力与速度、半径、周期的关系
- 典型例题：圆锥摆、汽车转弯
- 向心力与离心力的区别与联系
- 生活中的向心力应用（地球自转、洗衣机甩干、卫星轨道）
- 非匀速圆周运动分析

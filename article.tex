\section{引言}

在日常生活与工程实践中，\textbf{圆周运动}是一种极为常见的运动形式。无论是人造卫星绕地球运转、汽车转弯、游乐场的旋转设施，还是微观世界中电子绕原子核的运动，都离不开圆周运动。而在所有圆周运动的研究中，\textbf{向心力}（Centripetal Force）是最核心的概念之一。

本文将系统地介绍向心力的定义、来源、数学推导、典型应用场景，以及常见的认知误区。

\section{预备知识：极坐标与圆周运动}

在本文的分析中，我们将频繁使用极坐标这一有力工具。本节将从最基本的平面直角坐标出发，逐步推导极坐标下速度和加速度的表达式，确保读者能够完全理解后续章节的数学推导。

\subsection{极坐标基本概念}

在平面中确定一个点的位置，最直观的方式是使用\textbf{直角坐标}$(x,y)$：以原点 $O$ 为起点，沿 $x$ 轴方向移动 $x$ 单位，再沿 $y$ 轴方向移动 $y$ 单位。

然而，对于圆周运动，\textbf{极坐标}$(r,\theta)$ 更加方便：
\begin{itemize}
  \item $r$：\textbf{径向距离}（radial distance），即该点到原点 $O$ 的距离；
  \item $\theta$：\textbf{极角}（polar angle），即从正 $x$ 轴逆时针转过的角度（弧度制）。
\end{itemize}

两种坐标的转换关系为：
\[
x = r\cos\theta,\qquad y = r\sin\theta,\qquad
r = \sqrt{x^2+y^2},\qquad \theta = \arctan\frac{y}{x}.
\]

\subsection{单位矢量 $\hat{e}_r$ 与 $\hat{e}_\theta$}

在直角坐标系中，我们用 $\hat{i}$、$\hat{j}$ 表示沿 $x$、$y$ 轴的单位矢量。它们的大小为 1，方向固定不变。

极坐标中，我们定义两个新的单位矢量：
\begin{itemize}
  \item $\hat{e}_r$（径向单位矢量）：指向 $r$ 增大的方向，即从原点指向该点的方向；
  \item $\hat{e}_\theta$（横向单位矢量）：指向 $\theta$ 增大的方向，即 $\hat{e}_r$ 逆时针旋转 $90^\circ$ 的方向。
\end{itemize}

这两个矢量的大小均为 1，但它们的\textbf{方向随位置变化}（这是与 $\hat{i}$、$\hat{j}$ 的根本区别）。

用直角坐标的单位矢量表示：
\[
\hat{e}_r = (\cos\theta)\hat{i} + (\sin\theta)\hat{j},\qquad
\hat{e}_\theta = -(\sin\theta)\hat{i} + (\cos\theta)\hat{j}.
\]

可以验证：$\hat{e}_r \cdot \hat{e}_\theta = 0$（两者正交），且 $|\hat{e}_r| = |\hat{e}_\theta| = 1$。

\subsection{单位矢量的时间导数}

这是极坐标分析中最关键的一步。因为 $\hat{e}_r$ 和 $\hat{e}_\theta$ 的方向随 $\theta$ 变化，而 $\theta$ 可能随时间 $t$ 变化，所以它们对时间的导数不为零。

对 $\hat{e}_r = (\cos\theta)\hat{i} + (\sin\theta)\hat{j}$ 求导（注意 $\hat{i}$、$\hat{j}$ 是常矢量）：
\[
\frac{d\hat{e}_r}{dt} = (-\sin\theta)\dot{\theta}\,\hat{i} + (\cos\theta)\dot{\theta}\,\hat{j}
                      = \dot{\theta}\big[ -(\sin\theta)\hat{i} + (\cos\theta)\hat{j} \big]
                      = \dot{\theta}\,\hat{e}_\theta.
\]

同理，对 $\hat{e}_\theta = -(\sin\theta)\hat{i} + (\cos\theta)\hat{j}$ 求导：
\[
\frac{d\hat{e}_\theta}{dt} = -(\cos\theta)\dot{\theta}\,\hat{i} - (\sin\theta)\dot{\theta}\,\hat{j}
                           = -\dot{\theta}\big[ (\cos\theta)\hat{i} + (\sin\theta)\hat{j} \big]
                           = -\dot{\theta}\,\hat{e}_r.
\]

这两个结果极其重要，务必牢记：
\[
\boxed{\frac{d\hat{e}_r}{dt} = \dot{\theta}\,\hat{e}_\theta},\qquad
\boxed{\frac{d\hat{e}_\theta}{dt} = -\dot{\theta}\,\hat{e}_r}.
\]

其中 $\dot{\theta} \equiv \dfrac{d\theta}{dt}$ 表示角速度（Angular Velocity）。

\subsection{位置、速度、加速度的极坐标表达式}

\textbf{位置矢量：} 在极坐标中，位置矢量特别简单：
\[
\vec{r} = r\,\hat{e}_r.
\]

\textbf{速度：} 对位置求导：
\[
\vec{v} = \frac{d\vec{r}}{dt}
        = \frac{d}{dt}(r\hat{e}_r)
        = \dot{r}\,\hat{e}_r + r\,\frac{d\hat{e}_r}{dt}
        = \dot{r}\,\hat{e}_r + r\dot{\theta}\,\hat{e}_\theta.
\]

这就是极坐标下的速度公式。各项的物理意义：
\begin{itemize}
  \item $\dot{r}\,\hat{e}_r$：\textbf{径向速度}，描述该点沿半径方向远离或靠近原点的快慢；
  \item $r\dot{\theta}\,\hat{e}_\theta$：\textbf{横向速度}，描述该点绕原点转动的快慢。
\end{itemize}

\textbf{加速度：} 继续对速度求导：
\[
\vec{a} = \frac{d\vec{v}}{dt}
        = \frac{d}{dt}(\dot{r}\,\hat{e}_r + r\dot{\theta}\,\hat{e}_\theta)
        = \ddot{r}\,\hat{e}_r + \dot{r}\,\frac{d\hat{e}_r}{dt}
          + \dot{r}\,\dot{\theta}\,\hat{e}_r + r\ddot{\theta}\,\hat{e}_\theta
          + r\dot{\theta}\,\frac{d\hat{e}_\theta}{dt}.
\]

代入 $\dfrac{d\hat{e}_r}{dt} = \dot{\theta}\hat{e}_\theta$ 和 $\dfrac{d\hat{e}_\theta}{dt} = -\dot{\theta}\hat{e}_r$：
\[
\vec{a} = \ddot{r}\,\hat{e}_r + \dot{r}(\dot{\theta}\hat{e}_\theta)
        + \dot{r}\dot{\theta}\,\hat{e}_r + r\ddot{\theta}\,\hat{e}_\theta
        + r\dot{\theta}(-\dot{\theta}\hat{e}_r).
\]

合并 $\hat{e}_r$ 和 $\hat{e}_\theta$ 前的系数：
\[
\vec{a} = (\ddot{r} - r\dot{\theta}^2)\,\hat{e}_r
        + (r\ddot{\theta} + 2\dot{r}\dot{\theta})\,\hat{e}_\theta.
\]

这就是极坐标下加速度的完整表达式！各部分的物理意义：
\begin{itemize}
  \item $(\ddot{r} - r\dot{\theta}^2)$：\textbf{径向加速度}分量；
  \item $(r\ddot{\theta} + 2\dot{r}\dot{\theta})$：\textbf{横向加速度}分量。
\end{itemize}

其中 $-r\dot{\theta}^2$ 这一项尤为关键——它描述了由于方向变化（即使 $r$ 不变）而产生的"离心"效应，在圆周运动中它恰好提供了向心加速度。

\begin{center}
\fbox{$\displaystyle
\begin{aligned}
\vec{r} &= r\hat{e}_r, \\[4pt]
\vec{v} &= \dot{r}\hat{e}_r + r\dot{\theta}\hat{e}_\theta, \\[4pt]
\vec{a} &= (\ddot{r} - r\dot{\theta}^2)\hat{e}_r + (r\ddot{\theta} + 2\dot{r}\dot{\theta})\hat{e}_\theta.
\end{aligned}$}
\end{center}

\subsection{匀速圆周运动（特例）}

如果物体做\textbf{匀速}圆周运动（uniform circular motion），则：
\begin{itemize}
  \item 半径不变：$r = \text{常数}$，故 $\dot{r} = \ddot{r} = 0$；
  \item 角速度不变：$\dot{\theta} = \omega = \text{常数}$，故 $\ddot{\theta} = 0$。
\end{itemize}

代入上式，得到：
\[
\vec{v} = r\omega\,\hat{e}_\theta,\qquad
\vec{a} = -r\omega^2\,\hat{e}_r.
\]

注意到：
\begin{itemize}
  \item 速度方向沿 $\hat{e}_\theta$（切向），大小为 $v = r\omega$；
  \item 加速度方向沿 $-\hat{e}_r$（指向圆心），大小为 $a = r\omega^2 = \dfrac{v^2}{r}$——这正是向心加速度公式！
\end{itemize}

这说明向心加速度 $-r\omega^2\hat{e}_r$ 是极坐标加速度公式在匀速圆周运动这个特例下的自然结果，而非凭空引入的假设。

\subsection{非匀速圆周运动（$r$ 为常数，$\dot{\theta}$ 变化）}

如果物体做圆周运动但速率发生变化（$r$ 不变，但 $\dot{\theta}$ 变化），则：
\[
\dot{r} = \ddot{r} = 0,\qquad \ddot{\theta} \neq 0.
\]

代入通式：
\[
\vec{v} = r\dot{\theta}\,\hat{e}_\theta = r\omega\,\hat{e}_\theta,\qquad
\vec{a} = -r\dot{\theta}^2\,\hat{e}_r + r\ddot{\theta}\,\hat{e}_\theta.
\]

此时加速度有两部分：
\begin{itemize}
  \item $\vec{a}_n = -r\dot{\theta}^2\hat{e}_r$：\textbf{法向加速度}，只改变速度方向；
  \item $\vec{a}_t = r\ddot{\theta}\,\hat{e}_\theta$：\textbf{切向加速度}，只改变速度大小。
\end{itemize}

如果用线速度 $v = r\dot{\theta}$ 表示，则 $\dot{\theta} = v/r$，$\ddot{\theta} = \dot{v}/r$，上式可化为：
\[
\vec{a}_n = -\frac{v^2}{r}\hat{e}_r,\qquad
\vec{a}_t = \frac{dv}{dt}\,\hat{e}_\theta.
\]

这就是我们在正文中频繁使用的两个公式的来源。

\subsection{角动量的定义与物理意义}

\textbf{角动量}（Angular Momentum）是描述物体绕某一点旋转状态的物理量，在有心力场中具有特殊的重要性。

在平面运动中，角动量定义为
\[
L = rmv_\theta = mr^2\dot{\theta}.
\tag{1}
\]

其中 $r$ 是到参考点的距离，$v_\theta = r\dot{\theta}$ 是垂直于径向的速度分量。物理意义：角动量是"旋转的惯性"——质量越大、离参考点越远、转得越快，旋转状态就越难改变。

角动量的方向由右手定则确定：右手四指沿旋转方向弯曲，拇指指向的方向即为角动量方向。在本文涉及的平面圆周运动中，角动量始终垂直于运动平面，指向纸面外（为正）或纸面内（为负）。

在有心力场（如万有引力）中，力始终指向或背离中心天体，因此\textbf{合力对中心点的力矩为零}：
\[
\vec{M} = \vec{r} \times \vec{F} = 0,
\]
因为 $\vec{F}$ 与 $\vec{r}$ 共线（有力心但无偏心）。由刚体转动定理 $M = \dfrac{dL}{dt}$，立即得到
\[
\frac{dL}{dt} = 0 \quad\Longrightarrow\quad L = \text{常数}.
\]
这正是角动量守恒定律——它是时间平移对称性（物理定律不随时间变化）的直接推论。

角动量的量纲为 $[\text{能量}\times\text{时间}]$，在国际单位制中的单位是 $\text{kg}\cdot\text{m}^2/\text{s}$。

\subsection{矢量叉乘的几何意义}

在角动量的定义 $\vec{L} = \vec{r} \times m\vec{v}$ 中，涉及两个矢量的叉乘（外积）。理解叉乘的几何意义是掌握角动量概念的关键。

\textbf{叉乘的定义：} 两个三维矢量 $\vec{a}$ 和 $\vec{b}$ 的叉乘 $\vec{c} = \vec{a} \times \vec{b}$ 是一个新的矢量，其满足：

\begin{enumerate}
  \item \textbf{方向：} $\vec{c}$ 同时垂直于 $\vec{a}$ 和 $\vec{b}$，方向由右手定则确定——右手四指从 $\vec{a}$ 转向 $\vec{b}$（取小于 $180^\circ$ 的夹角），拇指指向即为 $\vec{c}$ 的方向；
  \item \textbf{大小：} $|\vec{c}| = |\vec{a}||\vec{b}|\sin\varphi$，其中 $\varphi$ 是 $\vec{a}$、$\vec{b}$ 之间的夹角。
\end{enumerate}

\textbf{几何直观：} $|\vec{a}||\vec{b}|\sin\varphi$ 恰好等于以 $\vec{a}$、$\vec{b}$ 为邻边所构成的平行四边形的面积。因此，叉乘的大小就是这两个矢量张成的平行四边形面积。

在平面运动中，$\vec{r}$ 和 $\vec{v}$ 均在运动平面内，它们的叉乘结果必然垂直于该平面——方向只有"穿出纸面"和"穿入纸面"两种可能。这与角动量守恒中"方向不变"的结论完全一致。

叉乘的一个重要代数性质是\textbf{反交换律}：
\[
\vec{a} \times \vec{b} = -(\vec{b} \times \vec{a}),
\]
这反映了左手系与右手系的区别。此外，当 $\vec{a}$ 与 $\vec{b}$ 共线（$\varphi = 0$ 或 $\pi$）时，$\sin\varphi = 0$，叉乘为零——这正是有心力场中力矩为零的原因，因为 $\vec{F}$ 与 $\vec{r}$ 共线。

叉乘在直角坐标分量下的计算公式（供读者参考）：
\[
\vec{a} \times \vec{b}
= \begin{vmatrix}
\hat{i} & \hat{j} & \hat{k} \\
a_x & a_y & a_z \\
b_x & b_y & b_z
\end{vmatrix}
= (a_y b_z - a_z b_y)\hat{i}
  + (a_z b_x - a_x b_z)\hat{j}
  + (a_x b_y - a_y b_x)\hat{k}.
\]

\section{什么是向心力？}

\subsection{向心力的定义}

\textbf{向心力}是使物体做圆周运动的一种力，它的方向始终指向圆心，与物体的运动方向（即切向方向）垂直。

\begin{center}
\fbox{向心力的方向：始终指向圆心 $\rightarrow$ 始终垂直于速度方向}
\end{center}

向心力并不是一种"新类型"的力，而是\textbf{各种已知力（如重力、弹力、摩擦力、电磁力等）的合力}在维持圆周运动时所表现出的效果。

\subsection{向心力不做功}

由于向心力始终与速度方向垂直，根据功的定义
\[
W = \vec{F}\cdot\vec{s} = Fs\cos\theta,
\]
当 $\theta=90^\circ$ 时，$\cos\theta=0$，因此\textbf{向心力对物体不做功}。这意味着向心力只改变运动方向，不改变速率。

\section{向心力公式推导}

\subsection{模型说明}

假设一质点在做匀速圆周运动，其速率为 $v$，速度为 $\vec{v}$。经过一个极短的时间 $\Delta t$ 后，物体走过一个极小的角度 $\Delta\theta$，速度变为 $\vec{v}+\Delta\vec{v}$。

结合上述说明，我们将根据加速度的定义
\[
\vec{a}=\frac{\Delta\vec{v}}{\Delta t}
\]
来分别推导向心加速度的方向和大小。

\subsection{向心加速度的方向推导}

根据 $\vec{a}=\frac{\Delta\vec{v}}{\Delta t}$ 可知，加速度的方向应与 $\Delta\vec{v}$ 的方向一致。

由于物体做匀速圆周运动，故有
\[
|\vec{v}| = |\vec{v}+\Delta\vec{v}| = v,
\]
即 $\vec{v}$、$\vec{v}+\Delta\vec{v}$、$\Delta\vec{v}$ 构成一个等腰三角形。

又根据几何关系可得，$\Delta\vec{v}$ 的对角亦为 $\Delta\theta$。注意 $\Delta\theta$ 是物体在 $\Delta t$ 时间内转过的角度，当 $\Delta t\to 0$ 时，$\Delta\theta\to 0$，此时 $\Delta\vec{v}$ 的方向将与 $\vec{v}$ 垂直，而加速度 $\vec{a}=\frac{\Delta\vec{v}}{\Delta t}$ 的方向也与 $\vec{v}$ 垂直。

也就是说，物体做匀速圆周运动时，\textbf{加速度方向与速度方向垂直，指向圆心}。

\subsection{向心加速度大小推导}

由向心加速度的定义：
\[
a = \frac{|\Delta\vec{v}|}{\Delta t}.
\]

其中 $|\Delta\vec{v}|$ 可由等腰三角形的几何关系经由\textbf{余弦定理}得到：
\[
\begin{aligned}
|\Delta\vec{v}|
&= \sqrt{|\vec{v}|^2 + |\vec{v}+\Delta\vec{v}|^2 - 2|\vec{v}||\vec{v}+\Delta\vec{v}|\cos\Delta\theta} \\[4pt]
&= \sqrt{v^2 + v^2 - 2v\cdot v\cdot\cos\Delta\theta} \\[4pt]
&= \sqrt{2}\,v\,\sqrt{1-\cos\Delta\theta}.
\end{aligned}
\]

此处用到二倍角公式：$1-\cos\Delta\theta = 2\sin^2\dfrac{\Delta\theta}{2}$，代入上式得：
\[
\begin{aligned}
|\Delta\vec{v}|
&= \sqrt{2}\,v\cdot\sqrt{2\sin^2\frac{\Delta\theta}{2}} \\[4pt]
&= 2v\sin\frac{\Delta\theta}{2}.
\end{aligned}
\]

当 $\Delta t$ 非常小时，$\Delta\theta$ 也非常小，此时 $\sin\dfrac{\Delta\theta}{2}\approx\dfrac{\Delta\theta}{2}$（等价无穷小替换）。代入上式得：
\[
\begin{aligned}
|\Delta\vec{v}|
&= 2v\cdot\frac{\Delta\theta}{2} \\[4pt]
&= v\cdot\Delta\theta \\[4pt]
&= v\cdot\omega\Delta t,
\end{aligned}
\]
其中 $\omega$ 为角速度，$\omega = \dfrac{v}{r}$，代入上式得：
\[
|\Delta v| = \frac{v^2}{r}\Delta t.
\]

因此
\[
\boxed{a = \frac{|\Delta\vec{v}|}{\Delta t} = \frac{v^2}{r}}.
\]

推导完毕。\qed

\subsection{结论}

物体做匀速圆周运动时，\textbf{加速度方向与速度方向垂直，指向圆心}；加速度大小为 $\dfrac{v^2}{r}$。

\subsection{补充说明}

\subsubsection{关于 $|\Delta\vec{v}|$ 的几何求法}

求 $|\Delta\vec{v}|$ 有更简单的办法——做等腰三角形底边的高，根据等腰三角形三线合一的性质，直接得出
\[
|\Delta\vec{v}| = 2v\sin\frac{\Delta\theta}{2}.
\]

\subsubsection{关于无穷小替换的严格条件}

将 $\sin\dfrac{\Delta\theta}{2}$ 替换为 $\dfrac{\Delta\theta}{2}$，这在微积分中称为\textbf{无穷小替换}。替换的依据为：

\begin{enumerate}
\item $\displaystyle\lim_{x\to 0}\frac{\sin x}{x}=1$（可理解为 $y=\sin x$ 与 $y=x$ 在 $x=0$ 处相切且斜率相同）
\item 被替换的项在方程中充当分子或分母中的\textbf{因式}
\end{enumerate}

若不满足以上条件贸然替换，可能产生错误结果。例如：
\[
A = \lim_{x\to 0}\frac{1}{\sin^2 x} - \frac{\cos^2 x}{x^2}.
\]
若直接替换 $\sin x\to x$：
\[
A \to \lim_{x\to 0}\frac{1}{x^2} - \frac{\cos^2 x}{x^2}
= \lim_{x\to 0}\frac{1-\cos^2 x}{x^2}
= \lim_{x\to 0}\frac{\sin^2 x}{x^2} = 1 \quad\text{（错误！）}
\]
正确做法是先通分，再在满足条件时使用替换：
\[
A = \lim_{x\to 0}\frac{x^2 - \sin^2 x\cos^2 x}{x^2\sin^2 x}
= \lim_{x\to 0}\frac{x^2 - \sin^2 x\cos^2 x}{x^4}
\;\longrightarrow\;
\lim_{x\to 0}\frac{x^2 - x^4}{x^4} = \lim_{x\to 0}\left(\frac{1}{x^2}-1\right) \to \infty.
\]

\section{向心力与其他物理量的关系}

线速度与周期 $T$ 的关系为：
\[
v = \frac{2\pi r}{T}.
\]

代入向心力公式：
\[
F_n = m\frac{(2\pi r/T)^2}{r} = \frac{4\pi^2mr}{T^2} = m\omega^2 r.
\]

频率 $f=1/T$，则 $\omega = 2\pi f$，故：
\[
F_n = 4\pi^2 mf^2 r.
\]

\section{典型例题分析}

\subsection{例题一：圆锥摆}

一根长度为 $L$ 的绳子一端固定，另一端系一个小球，使小球在水平面内做匀速圆周运动，绳子与竖直方向的夹角为 $\theta$。求小球的周期和绳子张力。

\textbf{分析：} 小球受两个力——重力 $mg$ 和绳子拉力 $T$。这两个力的合力提供向心力，方向水平指向圆心。

\textbf{受力分解：}
\[
\begin{cases}
T\cos\theta = mg & \text{（竖直方向平衡）}\\[6pt]
T\sin\theta = \dfrac{mv^2}{r} & \text{（提供向心力）}
\end{cases}
\]

由竖直方向平衡：$T = \dfrac{mg}{\cos\theta}$。

圆周半径：$r = L\sin\theta$。

代入向心力公式：
\[
T\sin\theta = \frac{mv^2}{L\sin\theta}
\quad\Longrightarrow\quad
v^2 = gL\sin\theta\cos\theta.
\]

周期 $T_{\text{摆}}$ 满足 $v = \dfrac{2\pi r}{T_{\text{摆}}} = \dfrac{2\pi L\sin\theta}{T_{\text{摆}}}$，解得：
\[
T_{\text{摆}} = 2\pi\sqrt{\frac{L\cos\theta}{g}}.
\]

\subsection{例题二：汽车转弯}

一辆质量为 $m=1200\,\text{kg}$ 的汽车，以速率 $v=36\,\text{km/h}=10\,\text{m/s}$ 通过一段水平弯道，弯道半径 $r=50\,\text{m}$。

\textbf{解：} 水平面上转弯所需的向心力由地面摩擦力 $f$ 提供：
\[
f = \frac{mv^2}{r} = \frac{1200\times 10^2}{50} = 24000\,\text{N}.
\]

竖直方向：$N = mg = 1200\times 9.8 = 11760\,\text{N}$。

若考虑最大静摩擦系数 $\mu=0.6$，最大静摩擦力：
\[
f_{\max} = \mu N = 0.6\times 11760 = 7056\,\text{N}.
\]
此时 $f>f_{\max}$，汽车将发生侧滑！因此车辆转弯时必须降速。

\section{向心力与离心力}

这是最容易混淆的概念。

\begin{itemize}
\item \textbf{向心力（Centripetal Force）}：\emph{真实存在的力}，来自其他物体（如绳子、桌面、重力等），方向指向圆心，是维持圆周运动的原因。
\item \textbf{离心力（Centrifugal Force）}：\emph{虚拟的惯性力}，只在非惯性参考系（如旋转车厢）中人为引入，方向背离圆心，大小等于 $m\omega^2 r$。
\end{itemize}

\textbf{记忆口诀：}
\begin{center}
\fbox{做圆周运动的物体\textbf{需要}向心力，不受\textbf{离心力}}
\end{center}

\section{生活中的向心力}

\subsection{地球自转与重力}

地球自转使得地球上的物体随地面一起旋转。这个旋转运动需要向心力，方向指向地轴。重力 $mg$（万有引力）提供向心力，因此：
\[
mg = F_{\text{万有引力}} - m\omega^2 r_{\perp}.
\]
其中 $r_{\perp}$ 为物体到地轴的垂直距离。这解释了：
\begin{itemize}
\item 同一物体在赤道上所受重力最小（$r_{\perp}$ 最大）
\item 在两极重力最大（$r_{\perp}=0$）
\end{itemize}

\subsection{洗衣机甩干原理}

衣物随滚筒高速旋转，水滴由于惯性保持原有的直线运动（而非随衣物做圆周运动），水滴通过衣物上的小孔被甩出，实现甩干效果。

\subsection{卫星轨道}

卫星绕地球做圆周运动的向心力由万有引力提供：
\[
\frac{GMm}{r^2} = m\frac{v^2}{r}
\quad\Longrightarrow\quad
v = \sqrt{\frac{GM}{r}}.
\]
这就是\textbf{第一宇宙速度}的来源（近地轨道 $v\approx 7.9\,\text{km/s}$）。

\section{非匀速圆周运动}

\subsection{从几何到动力学}

前面推导的基础是速度大小保持不变，即 $|\vec{v}|=|\vec{v}+\Delta\vec{v}|$。若速度大小也在变化，\textbf{速度矢量 $\vec{v}$ 的变化 $\Delta\vec{v}$ 将同时包含方向改变和大小改变两个来源}。

我们把 $\Delta\vec{v}$ 分解为两部分：
\[
\Delta\vec{v} = \Delta\vec{v}_{\text{方向}} + \Delta\vec{v}_{\text{大小}},
\]
其中 $\Delta\vec{v}_{\text{方向}}$ 仅由方向改变引起（大小不变），$\Delta\vec{v}_{\text{大小}}$ 仅由大小改变引起（方向不变）。

相应地，加速度也分解为两个分量：
\[
\vec{a} = \underbrace{\vec{a}_n}_{\text{法向加速度（改变方向）}}
        + \underbrace{\vec{a}_t}_{\text{切向加速度（改变速率）}}.
\]

将上式乘以质量 $m$，便得到合力的分解：
\[
\vec{F}_{\text{合}} = \underbrace{\vec{F}_n}_{\text{法向力（指向圆心）}}
                   + \underbrace{\vec{F}_t}_{\text{切向力（沿切线方向）}}.
\]

\subsection{法向加速度与切向加速度的大小}

\textbf{法向加速度 $a_n$ 的推导}完全沿用前文的结果——仅改变方向、不改变大小，故
\[
a_n = \frac{v^2}{\rho},
\]
其中 $\rho$ 为曲率半径。对于圆周运动，$\rho = r$，于是
\[
a_n = \frac{v^2}{r}.
\]

\textbf{切向加速度 $a_t$}来自速度大小的变化。设物体在 $\Delta t$ 时间内速率从 $v$ 变为 $v+\Delta v$，则
\[
a_t = \frac{\Delta v}{\Delta t} = \frac{dv}{dt}.
\]
方向沿切线，与速度同向（加速）或反向（减速）。

两个分量的方向互相垂直：
\[
\vec{a}_n \perp \vec{a}_t,\qquad |\vec{a}| = \sqrt{a_n^2 + a_t^2}.
\]

\subsection{极坐标推导（独立验证）}

为了交叉验证上述分解的正确性，我们用极坐标从运动学出发严格推导。

设物体在极坐标系 $(r,\theta)$ 中运动，其位矢为 $\vec{r}=r\,\hat{e}_r$，速度为
\[
\vec{v} = \dot{r}\,\hat{e}_r + r\dot{\theta}\,\hat{e}_\theta,
\]
其中 $\hat{e}_r$、$\hat{e}_\theta$ 为随物体转动的正交基矢。

对 $\vec{v}$ 求导，注意到 $\dot{\hat{e}}_r = \dot{\theta}\,\hat{e}_\theta$、$\dot{\hat{e}}_\theta = -\dot{\theta}\,\hat{e}_r$，得到加速度：
\[
\vec{a} = \bigl(\ddot{r} - r\dot{\theta}^2\bigr)\,\hat{e}_r
        + \bigl(r\ddot{\theta} + 2\dot{r}\dot{\theta}\bigr)\,\hat{e}_\theta.
\]

对于\textbf{圆周运动}（$r=$ 常数，即 $\dot{r}=0$），上式简化为
\[
\boxed{\vec{a} = -r\dot{\theta}^2\,\hat{e}_r + r\ddot{\theta}\,\hat{e}_\theta}.
\]

第一项 $-\dot{\theta}^2 r\,\hat{e}_r$ 是法向加速度，指向圆心，大小为
\[
a_n = \omega^2 r = \frac{v^2}{r}\quad(\omega=\dot{\theta},\; v=r\omega).
\]

第二项 $r\ddot{\theta}\,\hat{e}_\theta$ 是切向加速度，与切向方向一致，大小为
\[
a_t = r\alpha = r\frac{d\omega}{dt},
\]
其中 $\alpha=\ddot{\theta}$ 为角加速度。

由于 $v=r\omega$，对两边求导得
\[
\frac{dv}{dt} = r\frac{d\omega}{dt} = r\alpha,
\]
即 $a_t = \dfrac{dv}{dt}$，与几何分解完全一致。\qed

\subsection{动力学方程}

将加速度的两个分量代入 $\vec{F}=m\vec{a}$，得到非匀速圆周运动完整的动力学方程：

非匀速圆周运动完整的动力学方程为
\[
\begin{cases}
F_n = m\,\dfrac{v^2}{r}, & \text{法向力（提供向心加速度）}\\[8pt]
F_t = m\,\dfrac{dv}{dt}, & \text{切向力（提供切向加速度）}
\end{cases}
\qquad\Longrightarrow\qquad
\vec{F}_{\text{合}} = \vec{F}_n + \vec{F}_t.
\]

\subsection{法向力与切向力的物理意义}

这两个分量的物理作用完全不同：

\begin{itemize}
\item \textbf{法向力 $F_n$}：决定轨迹的弯曲程度。只改变速度方向，不改变速度大小。即使 $F_n=0$，物体仍可沿直线运动（$r\to\infty$ 的极限）。
\item \textbf{切向力 $F_t$}：决定速率的变化。只改变速度大小，不改变方向。即使 $F_t=0$，物体仍做匀速圆周运动。
\end{itemize}

当两者同时为零时，物体做匀速直线运动（静止或惯性运动）。

\section{非匀速圆周运动的典型场景}

\subsection{变半径圆周运动：弹簧振子}

一质量为 $m$ 的小球系在弹性系数为 $k$ 的轻弹簧一端，弹簧另一端固定在 $O$ 点。小球在光滑水平面上做往复运动，同时在水平面内旋转（弹簧长度随时间周期性变化）。设任意时刻弹簧长度为 $r(t)$，角速度为 $\omega(t)$。

\textbf{分析：} 小球受弹簧弹力 $F = -k(r-r_0)\,\hat{e}_r$（$r_0$ 为原长），方向始终沿半径方向，故为纯法向力（无切向分量）。

由极坐标加速度公式：
\[
\begin{cases}
-k(r-r_0) = m(\ddot{r} - r\omega^2) \quad\text{（径向）}\\[6pt]
0 = m(r\ddot{\theta} + 2\dot{r}\dot{\theta}) \quad\text{（切向）}
\end{cases}
\]
第二式给出角动量守恒条件 $r^2\dot{\theta} = \text{常数}$，第一式则描述径向的弹簧振动。

\subsection{变速圆锥摆：摆线运动}

圆锥摆中若绳子长度 $L$ 随时间变化（例如用滑轮收放绳子），则摆球同时做变速圆周运动。

\textbf{分析：} 设某一时刻绳子与竖直夹角为 $\theta$，长度为 $r$，角速度为 $\omega$。

竖直方向平衡（近似条件）：$T\cos\theta \approx mg$。

水平方向提供法向力：
\[
T\sin\theta = m\,\frac{v^2}{r} = m\,\frac{(r\omega)^2}{r} = m r\omega^2.
\]

若绳子被缓慢收缩（$dr/dt < 0$），根据角动量守恒 $L = mr^2\omega = \text{常数}$，收缩过程中 $\omega$ 增大，向心力需求增大，若 $T$ 不变，摆球将向内侧移动（$r$ 减小）以增大 $\omega$ 满足守恒。

\subsection{综合例题：汽车转弯并加速}

一辆质量 $m=1000\,$kg 的汽车，以速率 $v_0=10\,$m/s 驶入一段圆弧弯道（半径 $r=50\,$m）。驾驶员持续给油，使汽车沿弯道做加速运动，加速度大小恒为 $a_t=2\,$m/s$^2$（切向）。求：

（1）驶入弯道后第 3 秒时汽车所受合力的合力大小和方向；

（2）若汽车轮胎与地面间的最大静摩擦系数为 $\mu=0.8$，汽车不侧滑所能达到的最大切向加速度。

\textbf{解：}

（1）第 3 秒时，速率
\[
v = v_0 + a_t t = 10 + 2\times 3 = 16\;\text{m/s}.
\]

切向加速度 $a_t = 2\,$m/s$^2$（已知）。

法向加速度
\[
a_n = \frac{v^2}{r} = \frac{16^2}{50} = \frac{256}{50} = 5.12\;\text{m/s}^2.
\]

合力大小
\[
F = m\sqrt{a_n^2 + a_t^2}
   = 1000\times\sqrt{5.12^2 + 2^2}
   = 1000\times\sqrt{26.21 + 4}
   = 1000\times\sqrt{30.21}
   \approx 5496\;\text{N}.
\]

合力方向：设切向方向为正，则合力与切向的夹角 $\phi$ 满足
\[
\tan\phi = \frac{a_n}{a_t} = \frac{5.12}{2} = 2.56,
\qquad \phi = \arctan(2.56) \approx 68.6^\circ,
\]
即合力方向指向弯道内侧偏前方 $68.6^\circ$。

（2）汽车不侧滑的条件：法向力由摩擦力提供，最大值为 $f_n^{\max} = \mu mg = 0.8\times 1000\times 9.8 = 7840\,$N。

由法向动力学方程 $f_n = m v^2/r$，最大允许速率满足
\[
\frac{mv_{\max}^2}{r} \leq \mu mg
\quad\Longrightarrow\quad
v_{\max} \leq \sqrt{\mu g r} = \sqrt{0.8\times 9.8\times 50}
          = \sqrt{392} \approx 19.8\;\text{m/s}.
\]

在速率不超过 $19.8\,$m/s 的范围内，最大允许切向加速度为 $a_t^{\max} = 2\,$m/s$^2$（受发动机功率限制）。若弯道半径不变，且驾驶员全力加速，则当速率达到 $v_{\max}\approx 19.8\,$m/s 时，地面摩擦力刚好提供全部所需法向力，此时再加速将导致侧滑。

\section{角动量与动能定理}

非匀速圆周运动中，力对圆心的力矩为零时，角动量守恒；而法向力和切向力做功的情况截然不同。

\subsection{角动量守恒}

对圆心 $O$ 的角动量：
\[
\vec{L} = \vec{r}\times m\vec{v} = mr^2\dot{\theta}\,\hat{k}.
\]

合力对 $O$ 的力矩：
\[
\vec{M} = \vec{r}\times\vec{F}
        = \underbrace{\vec{r}\times\vec{F}_n}_{=0\text{（共线）}}
          + \underbrace{\vec{r}\times\vec{F}_t}_{=0\text{（平行）}}
        = 0.
\]
因此 $\vec{L} = mr^2\dot{\theta} = \text{常数}$，即 $r^2\dot{\theta} = \dfrac{L}{m}$ 为常数。

这一结论的物理意义：当只有法向力（指向圆心）和切向力（沿切线）作用时，合力的力矩始终为零，角动量必然守恒，与运动是否匀速无关。

从角动量守恒可推出一个重要推论：当 $r$ 减小时，$\dot{\theta}$ 必然增大，这正是花样滑冰收拢旋转时越转越快的原因。

\subsection{动能定理}

合力的功等于动能增量：
\[
W = \int \vec{F}_{\text{合}}\cdot d\vec{r}
  = \int F_t\,ds = \int m\,\frac{dv}{dt}\,v\,dt
  = m\int v\,dv = \frac{1}{2}mv^2\bigg|_{v_0}^{v}.
\]

其中 $ds = v\,dt$ 是弧微分。法向力 $F_n$ 永不做功（方向始终与位移 $d\vec{r}$ 垂直），只有切向力 $F_t$ 做功，这与我们关于"法向力只改变方向、只改变速度大小"的定性分析完全一致。\qed

\subsection{机械能守恒的条件}

若除重力、弹力等保守力外无其他外力做功，则机械能守恒。例如弹簧振子：
\[
\frac{1}{2}mv^2 + \frac{1}{2}k(r-r_0)^2 = E = \text{常数}.
\]
该方程与角动量守恒方程 $mr^2\dot{\theta} = L$ 联立，可完全确定弹簧振子的运动轨迹。

\section{统一视角：曲率与法向力}

回顾前文，法向加速度的通式为
\[
a_n = \frac{v^2}{\rho},
\]
其中 $\rho$ 是轨迹的曲率半径。对于圆周运动 $\rho=r$，对于任意平面曲线，$\rho$ 随位置变化。

\textbf{曲率半径的几何定义：} 在曲线上一点，曲率定义为 $\kappa = \dfrac{|d\theta|}{ds}$，其中 $d\theta$ 为切向方向转过的微元角，$ds$ 为对应的弧长微元。曲率半径 $\rho = 1/\kappa$。

以抛物线 $y = kx^2$ 为例：在顶点 $(0,0)$ 处，曲率为 $\kappa = |y''|/(1+y'^2)^{3/2} = 2k$，故 $\rho = 1/(2k)$。若物体沿该抛物线运动，在顶点处的法向加速度必为 $v^2/(1/(2k)) = 2kv^2$。

这个视角的意义在于：\textbf{法向力只由轨迹的几何形状决定，与速率的变化无关。} 无论速率是增加还是减少，只要物体沿同一轨迹运动，法向力都由 $F_n = mv^2/\rho$ 决定。

\section{从牛顿定律到开普勒三定律}

17世纪，丹麦天文学家开普勒根据第谷毕生的观测数据，总结出行星绕太阳运动的三大定律。两百多年后的今天，我们借助牛顿运动定律、万有引力定律以及完整的微积分工具，形式上严格地推导出这三大定律。整个推导仅需用到预备知识中的极坐标加速度公式，以及两个基本守恒定律——\textbf{角动量守恒}和\textbf{机械能守恒}。

\subsection{第一步：机械能守恒与角动量守恒}

行星在太阳引力 $F = -\dfrac{GMm}{r^2}$ 作用下运动，将其代入极坐标加速度的分量形式（见预备知识），由牛顿第二定律得：
\begin{align}
m\Bigl( \ddot{r} - r\dot{\theta}^2 \Bigr) &= -\frac{GMm}{r^2}, \tag{1a}\\
m\Bigl( r\ddot{\theta} + 2\dot{r}\,\dot{\theta} \Bigr) &= 0. \tag{1b}
\end{align}

\textbf{角动量守恒：} 方程 (1b) 可写为
\[
\frac{d}{dt}(r^2\dot{\theta}) = 0 \quad\Longrightarrow\quad
r^2\dot{\theta} = \frac{L}{m} = \text{常数}.
\]
由角动量定义 $L = mr^2\dot{\theta}$，这正是角动量守恒。

\textbf{机械能守恒：} 从 (1a) 中消去时间变量，利用 $\dot{r} = \dfrac{dr}{d\theta}\,\dot{\theta}$ 以及 $r^2\dot{\theta} = L/m$，经过配方法可得：
\[
\frac{1}{2}m\dot{r}^2 + \frac{1}{2}\frac{L^2}{mr^2} - \frac{GMm}{r} = E = \text{常数}.
\tag{2}
\]
方程 (2) 的物理意义清晰：左端第一项是径向动能，第二项是利用角动量表示的"离心势能"，第三项是引力势能，三者之和恒定。

注意到 $v_r = \dot{r}$、$v_\theta = r\dot{\theta}$，式 (2) 可简洁地写成
\[
\frac{1}{2}mv^2 - \frac{GMm}{r} = E,
\]
这正是行星运动的机械能守恒定律。

\subsection{第二步：开普勒第二定律（面积定律）}

由角动量守恒，立即得到掠面速率：
\[
\frac{dA}{dt} = \frac{1}{2}r^2\dot{\theta} = \frac{L}{2m} = \text{常数}.
\tag{3}
\]
其中 $A$ 是行星与太阳连线扫过的面积。由于 $\dfrac{dA}{dt}$ 为常数，行星与太阳的连线在相等时间内扫过相等的面积——开普勒第二定律得证。

\textbf{几何直观：} 行星靠近太阳时（$r$ 小），线速度必须增大以保持面积速度不变；远离太阳时（$r$ 大），速度减小。这正是椭圆轨道上行星近日点快、远日点慢的根本原因。

\subsection{第三步：开普勒第一定律（轨道定律）}

现在从 (2) 出发，消去时间变量 $t$，得到 $r$ 随 $\theta$ 变化的轨道方程。

由 (3) 有 $\dot{\theta} = \dfrac{L}{mr^2}$，代入 (2) 并利用链式法则 $\dot{r} = \dfrac{dr}{d\theta}\,\dot{\theta}$，得：
\[
\frac{dr}{d\theta} = \frac{mr^2}{L}\sqrt{\frac{2E}{m} + \frac{2GM}{r} - \frac{L^2}{m^2r^2}}.
\tag{4}
\]

令 $u = \dfrac{1}{r}$，则 $\dfrac{dr}{d\theta} = -\dfrac{1}{u^2}\dfrac{du}{d\theta}$，代入并整理为关于 $u(\theta)$ 的方程：
\[
\left( \frac{du}{d\theta} \right)^2 = \frac{G^2M^2m^4}{L^4}\left( 1 + \frac{2EL^2}{G^2M^2m^3} \right) - \left( u - \frac{GMm^2}{L^2} \right)^2.
\tag{5}
\]

对 (5) 积分（利用反正弦函数的原函数），得到：
\[
u = \frac{1}{r} = \frac{GMm^2}{L^2}\bigl[ 1 + \epsilon\cos(\theta + \theta_0) \bigr],
\tag{6}
\]
其中 $\theta_0$ 是初相，$\epsilon$ 是\textbf{偏心率}（eccentricity）：
\[
\epsilon = \sqrt{1 + \frac{2EL^2}{G^2M^2m^3}}.
\tag{7}
\]

取适当的极轴方向使 $\theta_0 = 0$，则轨道方程为：
\[
\boxed{r = \frac{p}{1 + \epsilon\cos\theta}},\qquad
p = \frac{b^2}{a},\quad \epsilon = \frac{c}{a}.
\tag{8}
\]

方程 (8) 正是圆锥曲线的极坐标标准方程。根据能量 $E$ 的符号：

\begin{itemize}
  \item $E < 0,\; \epsilon < 1$：轨道为\textbf{椭圆}——行星轨道（开普勒第一定律）；
  \item $E = 0,\; \epsilon = 1$：轨道为\textbf{抛物线}；
  \item $E > 0,\; \epsilon > 1$：轨道为\textbf{双曲线}的一支。
\end{itemize}

\textbf{验证椭圆性：} 由 (8) 可推出直角坐标下的标准椭圆方程。设太阳位于左焦点 $F_1(-c,0)$，远地点到右焦点 $F_2(c,0)$ 的距离满足
\[
r + r' = 2a \quad (\text{常数}),
\]
即到两焦点距离之和恒定的点的轨迹——这正是椭圆的定义。

\subsection{第四步：椭圆轨道特殊点的速率}

行星椭圆轨道（$E<0,\;\epsilon<1$）的几个特殊点具有简洁的速度表达式。设近日点（距太阳最近）与远日点（距太阳最远）处的速率分别为 $v_1$、$v_2$，则由能量守恒与角动量守恒：
\[
\begin{cases}
\dfrac{1}{2}mv_1^2 - \dfrac{GMm}{a-c} = \dfrac{1}{2}mv_2^2 - \dfrac{GMm}{a+c},\\[8pt]
m(a-c)v_1 = m(a+c)v_2,
\end{cases}
\]
解得：
\[
v_1 = \frac{a+c}{b}\sqrt{\frac{GM}{a}},\qquad
v_2 = \frac{a-c}{b}\sqrt{\frac{GM}{a}}.
\tag{9}
\]

短轴顶点处的速率更为简洁（此处 $r=a$，$v_\theta = v$，$v_r = 0$）：
\[
v_3 = \sqrt{\frac{GM}{a}}.
\tag{10}
\]

由 (10) 立即得到能量与角动量的轨道参数表示：
\[
E = \frac{1}{2}mv_3^2 - \frac{GMm}{a} = -\frac{1}{2}\frac{GMm}{a},
\qquad
L = m v_3 b = mb\sqrt{\frac{GM}{a}}.
\tag{11}
\]

值得注意的是：\textbf{轨道上各点的速率完全由 $a$ 决定，与行星质量 $m$ 无关。} 这正是 (9)(10) 两式的核心含义。

\subsection{第五步：开普勒第三定律（周期定律）}

联立掠面速率 (3) 和短轴顶点速率 (10)：
\[
\frac{dA}{dt} = \frac{L}{2m} = \frac{b}{2}\sqrt{\frac{GM}{a}}.
\tag{12}
\]

对上式从 $0$ 到 $T$ 积分（$T$ 为周期），行星恰好扫过整个椭圆，面积 $A = \pi ab$：
\[
\pi ab = \frac{b}{2}\sqrt{\frac{GM}{a}}\,T
\quad\Longrightarrow\quad
\boxed{\frac{a^3}{T^2} = \frac{GM}{4\pi^2} = \text{常数}}.
\tag{13}
\]

这正是开普勒第三定律！常数 $K = \dfrac{GM}{4\pi^2}$ 仅取决于中心天体（太阳）的质量，与行星本身无关。

三大定律的统一证明至此完成。整个推导链条如下：

\[
\text{牛顿第二定律 + 万有引力}
\;\xrightarrow{\text{极坐标分解}}\;
\begin{cases}
\text{角动量守恒} \;\Longrightarrow\; \text{开普勒第二定律} \\
\text{机械能守恒} \;\Longrightarrow\; \text{开普勒第一定律（轨道方程）}
\end{cases}
\;\xrightarrow{\text{联立求解}}\;
\text{开普勒第三定律}.
\]

\subsection{延伸：三种宇宙速度}

由能量方程 $\dfrac{1}{2}mv^2 - \dfrac{GMm}{r} = E$ 可以推导出几个重要的临界速度。

\textbf{第一宇宙速度（环绕速度）：} 使物体在地球表面做圆轨道运动所需最小速度：
\[
v_1 = \sqrt{\frac{GM_\oplus}{R_\oplus}} \approx 7.9\;\text{km/s}.
\]

\textbf{第二宇宙速度（逃逸速度）：} 使物体刚好能逃逸到无穷远（$r\to\infty,\;v\to 0$），即 $E=0$ 时的速度：
\[
v_2 = \sqrt{\frac{2GM_\oplus}{R_\oplus}} = \sqrt{2}\,v_1 \approx 11.2\;\text{km/s}.
\]

\textbf{第三宇宙速度：} 使物体从地球表面逃逸太阳系所需速度（考虑地球公转的"引力协助"）：
\[
v_3 \approx 16.7\;\text{km/s}.
\]

这三个速度构成了人类航天能力的基本门槛——从环绕地球（$v_1$）、到登陆月球（$v_2$）、再到飞出太阳系（$v_3$），背后都是能量守恒与角动量守恒的精确计算。
